## Instalación Microsoft 365 (Versión de suscripción)

1. Descarga manifest.xml
2. Abre Word
3. Insertar → Complementos → Mis complementos
4. Agregar desde archivo
5. Selecciona manifest.xml

En Word 2019 (y 2016), Microsoft diseñó el sistema pensando en empresas. Para cargar un XML localmente, te obligan a hacer el "Sideloading" por Carpeta Compartida:
Tienes que poner el XML en una carpeta.
Compartir esa carpeta en Windows (propiedades > Compartir).
En Word, ir a Opciones > Centro de Confianza > Catálogos de complementos de confianza y pegar la ruta de red de esa carpeta.
Reiniciar Word.
